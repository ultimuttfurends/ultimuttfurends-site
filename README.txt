UltiMutt Furends website

How to use:
1. Open index.html in your browser.
2. Replace placeholder text/photos.
3. For the contact form, create a free Formspree form and replace:
   https://formspree.io/f/your-form-id
4. Upload the folder to Netlify, Cloudflare Pages, or GitHub Pages.

Recommended no-hosting-fee option:
- Netlify: easiest drag-and-drop deployment with free custom domain + SSL.
